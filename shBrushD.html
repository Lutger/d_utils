Syntax highlighting D code with javascript

This script for syntax highlighting D code is to be used with the SyntaxHighlighting library. It is quite easy to set up, see the following links for more information:
  • installation
  • configuration
There are some limitations to the current D brush - such as nested comments - but for most snippets it should be more than adequate. The D brush can be downloaded from its github repository.
